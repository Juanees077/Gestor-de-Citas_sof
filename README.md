# Gestor de Citas SOF

Aplicacion web gratuita para agendar citas de una profesional independiente. Esta pensada para bajo volumen: entre 1 y 6 clientes por dia, y hasta 25 clientes durante fines de semana.

## Funcionalidades

- Cliente: agenda citas segun disponibilidad, consulta horarios libres y recibe recordatorio.
- Administradora: ve calendario semanal, crea citas manuales, edita, elimina y envia recordatorios.
- Configuracion: servicios, precios, duracion, horario laboral, aviso minimo y datos del negocio.
- Control de conflictos: una sola cita activa por franja horaria.
- Despliegue gratis: busca la forma de que todo el sistema se gratis que no gaste un peso en nada
